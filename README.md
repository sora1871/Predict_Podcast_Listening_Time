# Predict_Podcast_Listening_Timeの分析

このプロジェクトは、Podcastのエピソードに対して視聴時間を予測するモデルを構築することを目的としています。

## 🚀 概要

- Podcastのエピソードに関する情報を用いて、リスナーがどれくらいの時間視聴するかを予測します。
- Kaggleのコンペティションで提供されたデータを使用しています。

## 🛠 使用技術・ライブラリ

- Python
- pandas
- numpy
- scikit-learn
- LightGBM
- XGBoost

## 📊 データセット

- 提供元: Kaggle
- ファイル: `train.csv`, `test.csv`
